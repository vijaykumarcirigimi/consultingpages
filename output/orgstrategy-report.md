## Notes

- **Full coverage achieved.** The updated library now covers all 16 content sections with no design gaps.
- **Differentiator variety achieved.** Section 5 (Why It Matters) uses `diff-image-list` and section 11 (Why Edstellar) now uses `stats-icon-cards` — eliminating the visual repetition from the previous report.
- **Sub-services card count:** Content has 8 training programs but the template supports 6 (3x2). Extend to 3x3 to accommodate all 8 programs (or 9 with the custom program CTA as a card).
- **FAQ question count:** Content has 8 FAQs but the template defaults to 5. The accordion pattern extends naturally — duplicate the markup.
- **Process step count:** Content has 4 steps; the vertical stepper template supports 5. Remove one placeholder step or keep it compact at 4.
- **SEO metadata template:** The `seo-metadata-block-template.md` is a markdown developer handoff template, not an HTML design. Pass to the dev team with the per-page values from the content file's SEO block (slug, meta title, meta description, keywords, H1).
