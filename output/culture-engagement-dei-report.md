## Notes
- The 3 service areas (Culture Audit, Employee Engagement, DEI Diagnostics) are combined into a single tabbed section using the outcomes-horizontal-tabs design. Each tab gets the service heading, description, and bullet points on the left, with the corresponding data visualization (Culture Health Scorecard, Engagement Driver Analysis, DEI Maturity Scorecard) built as rich HTML embeds on the right panel. Only 3 of the 6 available tab slots are used.
- The data visualizations require custom rich HTML embeds: a heatmap-style scorecard for Culture Audit, a bubble/priority chart for Engagement Driver Analysis, and a horizontal bar maturity chart for DEI Diagnostics — all using the existing brand colour system (navy, lime, red for critical indicators).
- Connected services section uses the scrollable navy variant to accommodate all 6 cross-sell links without vertical bloat.
- Page flow alternates backgrounds: light hero → navy stats → navy logo wall → light challenges → light outcomes tabs → navy process → light testimonial → navy download → navy connected → light FAQ → light form → navy footer. Consecutive navy sections (logo wall → challenges transition, and download → connected) are acceptable since they serve different content purposes and break visually.
- Single testimonial works in the carousel by displaying as a static card — no navigation dots needed.
