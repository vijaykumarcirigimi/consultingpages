## Setup — Step by step

### Step 1: Create the folder on your machine

Create this folder wherever you want (I see you have `Documents/Consulting Page Builder`):

```
C:\Users\[YourName]\Documents\Consulting Page Builder\
```

### Step 2: Copy the files

Download all files from this Claude conversation and organize them:

1. **Move all HTML files** you downloaded from our sessions into the `library/` folder
2. **Copy** `library-index.json` into the `library/` folder
3. **Copy** `skills/design-scout/SKILL.md` into `skills/design-scout/`
4. **Copy** `agents/design-scout-agent.md` into `agents/`
5. **Copy** `content/l-and-d-consulting.md` into `content/`

### Step 3: Set up Claude Code

If you don't have Claude Code installed yet:

```bash
npm install -g @anthropic-ai/claude-code
```

Then navigate to your project folder:

```bash
cd "C:\Users\[YourName]\Documents\Consulting Page Builder"
```

### Step 4: Run the Design Scout agent

```bash
# Option 1: Full command
claude "Read skills/design-scout/SKILL.md for instructions. Then analyze content/l-and-d-consulting.md against library/library-index.json and generate report.md mapping each section to the best available HTML design. Flag any sections that need new designs."

# Option 2: Shorter version
claude "You are the Design Scout agent. Use skills/design-scout/SKILL.md. Analyze content/l-and-d-consulting.md and output report.md"
```

### Step 5: Check the output

The agent will generate `report.md` in the project root with:
- Section-by-section design mapping
- Gap analysis (sections needing new designs)
- Recommended page flow
- Notes for your design team

## Adding new content files

For each new consulting page, create a markdown file in the `content/` folder:

```
content/
├── l-and-d-consulting.md
├── talent-assessment.md          ← New page
├── leadership-development.md     ← New page
├── change-management.md          ← New page
└── compliance-training.md        ← New page
```

Each content file should list all the sections with their headings and content. See `l-and-d-consulting.md` as a template.

Then run the Design Scout on the new file:

```bash
claude "Design Scout: analyze content/talent-assessment.md against library/library-index.json and output report.md"
```

## Adding new HTML designs to the library

When you create new section designs:

1. Save the HTML file in the `library/` folder
2. Update `library/library-index.json` to add the new variant under the correct section category
3. If it's an entirely new section type, add a new category to the JSON
